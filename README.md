# SQL Visual Guide

A visual, beginner-friendly SQL learning repository designed to simplify core and commonly confusing topics through infographic-style slides.

The focus of this project is **visual understanding of SQL concepts**, not memorization of syntax.

---

## Purpose

The goal of this repository is to make key SQL concepts easier to understand by presenting them in a clear and visual format.

This project is useful for:

- beginners learning SQL fundamentals  
- junior analysts building intuition  
- students creating portfolio projects  
- visual learners who prefer diagrams over long explanations

## Topics Covered

### Joins
How tables are connected and how different join types affect results.

### Group By vs Window Functions
Difference between aggregation and row-level calculations.

### Data Cleaning
Basic techniques for handling messy or inconsistent data.

### Common Table Expressions (CTE)
Breaking complex queries into readable logical steps.

### Stored Procedures
Encapsulating logic inside reusable database routines.

### Indexes
Understanding how indexes improve query performance.

---

## Learning Approach

Each topic is presented as a visual guide to help understand the logic first.

Typical flow:

1. review the infographic  
2. understand the idea  
3. explore SQL examples (coming soon)
